# 正面对比总表（以舟山出发为例）

<div class="mt-6">

| 维度 | 舟山汽运小程序 | 第三方平台 |
| :--- | :--- | :--- |
| 购票服务费 | **0 元** | 3–10 元/单，不退 |
| 默认搭售 | 无 | 保险 10–30 元（需手动取消） |
| 2h 前退票费 | 票面 **10%** | 票面 20% + 服务费不退 |
| 2h 内退票费 | 票面 **20%** | 票面 25% + 服务费不退 |
